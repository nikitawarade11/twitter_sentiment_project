# twitter_sentiment_project
# 📊 Real-Time Twitter Sentiment Analysis

## 🔍 Overview
This project analyzes real-time Twitter data to understand public sentiment on trending topics. It uses Natural Language Processing (NLP) techniques to classify tweets into Positive, Negative, or Neutral categories.

---

## 🚀 Features
- Fetches real-time tweets using Twitter API  
- Cleans and preprocesses text data  
- Performs sentiment analysis using VADER  
- Displays sentiment distribution  

---

## 🛠️ Tech Stack
- Python  
- Tweepy  
- NLTK (VADER)  
- Pandas  

---

## ⚙️ Workflow
1. Collect tweets using keywords/hashtags  
2. Clean text (remove URLs, mentions, hashtags)  
3. Apply sentiment analysis  
4. Classify sentiment  
5. Visualize results
